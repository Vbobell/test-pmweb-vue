# test-pmweb-vue
Repository created to test pmweb with vue.

## Required installation tools

* [node](https://nodejs.org/en/)
* [yarn](https://yarnpkg.com/en/docs/install#debian-stable)

## tools

* babel
* webpack
* axios
* Vue
* Sass
* highcharts

## Instalation
```bash
yarn install
```

## Run
```bash
yarn start
```

## Access 
```bash 
localhost:3000 or 127.0.0.1:3000
```
